# Medical Insurance Cost Analysis

## Objective
To analyze factors affecting medical insurance charges and derive meaningful insights using data analysis and visualization.

## Tools Used
- Python (Pandas, Matplotlib)
- Google Colab
- Looker Studio (Dashboard)

## Dataset
Medical Insurance Dataset (Kaggle)

## Key Analysis
- Data Cleaning (handled duplicates, validated dataset)
- Exploratory Data Analysis (EDA)
- Feature Engineering (BMI Category)
- Data Visualization (Charts & Dashboard)

## Key Insights

- Smokers pay 3.8× more — $32K vs $8.4K for non-smokers. 
- Older age moderately increases charges — rises with each decade.
- Obese individuals pay more, especially when combined with smoking.
- More children slightly correlates with higher insurance costs.
- Lifestyle factors (e.g., smoking, BMI) have a stronger impact than demographic factors (e.g., gender, region).

Lifestyle factors such as smoking have a major impact on insurance costs, while demographic factors show relatively lower influence.
